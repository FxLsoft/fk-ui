通过设置 `direction="vertical"` 可以展示竖直的单选框组。

---


```vue { "component": true } 
<template>
  <fk-radio-group direction="vertical">
    <fk-radio value="A">A</fk-radio>
    <fk-radio value="B">B</fk-radio>
    <fk-radio value="C">C</fk-radio>
    <fk-radio value="D">D</fk-radio>
  </fk-radio-group>
</template>
```
