标签的大小分为：`small`、`medium`、`large` 三种，可以在不同场景下选择合适按钮尺寸。推荐及默认尺寸为 `medium`。

---


```vue { "component": true } 
<template>
  <fk-space>
    <fk-tag size="large">Large</fk-tag>
    <fk-tag>Medium</fk-tag>
    <fk-tag size="small">Small</fk-tag>
  </fk-space>
</template>
```
